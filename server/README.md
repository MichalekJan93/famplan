## Description
Server part of the application, using Mongoose, Express and MongoDB

## Installation
```
npm init --yes
npm install -g nodemon
npm install dotenv
npm install mongoose@~6.6.0
npm install express@~4.18.1
npm install joi@~17.6.0
npm install cors
npm i bcrypt@~5.1.0
npm i express-session@~1.17.3
npm install axios
```
